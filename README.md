# Install WordPress in Docker with Nginx and Lets Encrypt for HTTPS And Also React and Express

## Install
* Clone from Github
* Yarn
    `yarn && cd client && yarn`

## Start
* Start WordPress
    `docker-compose up -d`
* Start Node Client and Server
    `yarn start`
